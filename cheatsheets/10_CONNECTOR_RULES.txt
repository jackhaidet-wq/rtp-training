THE THREE-WORD RULE
  Read. Write. Approval.

WHAT EACH MEANS
  READ      Claude can pull data from the connector.
  WRITE     Claude can change something in the connector.
  APPROVAL  Claude shows you the action and waits before
            executing.

THE APPROVAL PATTERN
----------------------------------------------------------------
  When Claude wants to call a connector tool, you see:
    - WHICH connector
    - WHICH tool / action
    - WHAT data it would read or write
    - APPROVE / DENY

  Default to denying anything that surprises you.
